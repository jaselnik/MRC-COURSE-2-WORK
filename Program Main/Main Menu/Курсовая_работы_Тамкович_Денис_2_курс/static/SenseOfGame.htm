Головоломка "Неприкасаемое судоку"


Неприкасаемое судоку - настольная логическая игра на квадратном поле размером 8x8 (9х9) клеток для одного участника.

Ваша цель - Заполните сетку числами от 1 до n (n - размер сетки) так, чтобы во всех строках, столбцах и выделенных областях числа были различны. Клетки с одинаковыми числами не должны касаться друг друга.

В начале даётся поле 8х8(9х9) с одним из уровней сложности (НОВИЧОК, ЛЮБИТЕЛЬ, ПРОФЕССОР), на этом поле даются ячейки, которые либо заполнены цифрой, либо пустые и 8(9) блоков разных цветов.

Игра завершается в том случае, когда игрок решает правильно головоломку либо самостоятельно прерывает игру.
